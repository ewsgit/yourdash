# YourDash Desktop Client

An application created using the YourDash WebClient and UIKit and allow use of PSA and integrate better with user's workflows

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
